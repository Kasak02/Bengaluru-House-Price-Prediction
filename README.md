# 🏠 Bengaluru House Price Prediction

A Machine Learning web application that predicts the **price of houses in Bengaluru** based on property features such as location, BHK, bathrooms, balconies, and total square feet.

---

## 🚀 Live Demo

🔗 **Deployed Application:**
https://kasak02-bengaluru-house-price-prediction-app-gjke85.streamlit.app/

Try entering house details and get an **instant price prediction**.

---

## 🚀 Features

* 📍 Select house **location**
* 🛏 Choose number of **BHK**
* 🚿 Enter number of **bathrooms**
* 🌇 Enter number of **balconies**
* 📏 Input **total square feet**
* 💰 Get **instant predicted house price**

---

## 🛠 Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Streamlit

## ▶️ Run Locally

Clone the repository

```
git clone https://github.com/Kasak02/bengaluru-house-price-prediction.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the app

```
streamlit run app.py
```
